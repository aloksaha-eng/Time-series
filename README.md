# 📈 Netflix Stock Price Forecasting

This project analyzes historical Netflix (NFLX) stock prices from **2018–2022** and forecasts future prices using the **ARIMA (AutoRegressive Integrated Moving Average)** time series model. The workflow includes data preprocessing, stationarity testing with the Augmented Dickey-Fuller (ADF) test, differencing, ACF/PACF analysis, model building, and forecasting.

## 🚀 Features

* Historical stock price analysis
* Data preprocessing and visualization
* Stationarity testing using the ADF test
* ACF and PACF plots for parameter selection
* ARIMA model training and forecasting
* Time series trend visualization

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Statsmodels
* Jupyter Notebook

## 📊 Objective

The goal of this project is to apply time series analysis techniques to model historical Netflix stock prices and generate future forecasts, demonstrating practical forecasting methods used in financial data analysis.

## 📁 Dataset

* **Company:** Netflix (NFLX)
* **Time Period:** 2018–2022
* **Target Variable:** Adjusted Closing Price (`Adj Close`)
